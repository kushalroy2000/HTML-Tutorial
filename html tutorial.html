HTML Tags:

HTML:

Defines the root of the HTML document.

HEAD:

Contains metadata/information of the document.

BODY:

Body contains the entire HTML content.

UL:

Defines an unordered list.

OL:

Defines an ordered list.

WBR:

Defines a possible line-break.

BR:

BR use for page breaking.

TD:

Defines a cell in the table.

TR:

Defines a row in table.

STYLE:

Defines style information for a document.
